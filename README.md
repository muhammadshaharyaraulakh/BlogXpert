# BlogXpert
BlogXpert is a role-based blogging platform built with HTML, CSS, JavaScript, PHP, and MySQL. It features three user roles for streamlined content management.

# Readers 
Can browse approved posts, leave comments, and like articles.
# Writers 
Can create, edit, and submit blog posts for admin approval. Users can apply to become writers via an application form.
# Admins
Have full control: they approve or delete posts, manage categories.
# Main Admin
Will Assign roles (adding/deleting writers and admins).

The platform includes secure login, session management, and dedicated dashboards for each role. The frontend is fully responsive, providing a seamless experience across all devices.

Key Features:

Role-specific dashboards

Blog post creation, editing, and approval workflow

Interactive like and comment system

Categories management

User role application and management

Tech Stack: Frontend: HTML5, CSS3, JavaScript | Backend: PHP | Database: MySQL
